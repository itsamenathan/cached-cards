# Cached Cards

Cached Cards is an offline-first Progressive Web App (PWA) for browsing card game rules.

## Features
- Offline-ready library with rules stored as Markdown + frontmatter
- Real-time search and player-count filtering
- Installable PWA with service worker caching

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
